PROGRAMMING LANGUAGES

LOW LEVEL LANGUAGES

These are classified as low level because they can easily be understood by the computer directly.They are hardware dependants for exaple a program designed for one computer can not be installed and used on another

MACHINE LANGUAGE

These are languages in which programming is done in binary logic for example instrutions are written in 0s and 1s

They are machine dependant for example they vary from computer to computer

HIGH LEVEL LANGUAGES

Languages that are closer to human language but they need language processor to work backwards

A high level language resembles some human language like English

ASSEMBLY PROGRAMMING LANGUAGES

These allow users to write a program using abbreviations or even more easily remembered words

They need assemblers to translate them to machine language,understable by the computer
